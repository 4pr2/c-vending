Kомплексное решение для онлайн мониторинга торговых аппаратов

C-Vending


Система предоставляет возможность просматривать по каждому аппарату текущее состояние, наличие ингредиентов, монет, купюр, проведенных инкассаций. Так же в системе предусмотрено возможность удаленного управления аппаратом (изменение цен, отключение аппарата и др.).

Для аппаратной части существует технология «Vendisp» которая считывает данные с индекатора вендингового аппарата и купироприемника. Что позволяет улучшить объем собираемых данных о процессах происходящих в аппарате.


Отчеты

Позволяет формировать отчеты по работе торговых автоматов, с заданием временного интервала, конкретных автоматов или групп, договоров аренды, обслуживающих автоматы сотрудников и многих других параметров.

Оповещение

В случае ошибок на торговом автомате информация об этом мгновенно приходит в систему или на указанный номер телефона.

Отвечающий за работу автомата сотрудник оперативно отреагирует на событие.

Безопасность

Данные передаются по защищенным каналам связи стандарта GSM/GPRS. Доступ пользователей к системе осуществляется только по логину и паролю.

C-Vending гарантирует сохранность и конфиденциальность.


Расширение функционала торговых аппаратов


Для модуля CV-2 существует возможность подключения функции списания остатка (сдачу) средств клиента на его персональный счет посредством RFID карты.

В дальнейшем, с помощью накопленых средств на карте, клиент может рассчитываться в ваших вендинговых аппаратах.



CV-1

Считывание модемом CV-1 данных аудита с вендингового аппарата в формате EVA-DTS и передает по сетям GSM, GPRS в Программу мониторинга.

CV-2

CV-2 это решения для производителей вендинговых аппаратов, представляет собой полностью готовою электронную начинку для вендингового аппарата.
